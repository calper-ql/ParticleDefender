# ParticleDefender
A little interactive program for my AR3200 class

## Requirements:
you need glfw installed on your computer with OpenGL 3.3 and higher

## Build
	mkdir build
	cd build
	cmake ..
	make

### Programs
	test_v 1 and 2 are just for testing.
	test_v3 is for show :)
